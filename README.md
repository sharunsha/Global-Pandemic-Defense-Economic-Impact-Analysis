Global Pandemic Impact Analysis (Tableau Project)
Overview
This project analyzes how different countries were impacted during global pandemic conditions.
Using data collected from WHO, IMF, World Bank, and Our World in Data (OWID), the study visualizes:
GDP loss across countries
Medical spending patterns
Defense level vs. economic loss
Global risk distribution
Country-level COVID statistics (cases, deaths, survival rate, infection ratio, etc.)
Two interactive Tableau dashboards were designed to explore these insights visually.
